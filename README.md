<h2 style="color: #ffffff; border-bottom: 2px solid #334155; padding-bottom: 10px;">⚡ Local Interface Initialization</h2>
<p style="color: #cbd5e1; font-size: 1.05em;">The client dashboard operates completely independent of heavy compile runtimes. Initialize your layout tree using these simple steps:</p>

<ol style="color: #cbd5e1; line-height: 1.8;">
    <li>Clone the client codebase structure locally: <code style="color: #38bdf8; background: #1e293b; padding: 3px 8px; border-radius: 6px;">git clone https://github.com/your-repo/compliance-ai-frontend.git</code></li>
    <li>Ensure the configuration entry point matches your server bridge. Open <code style="color: #a78bfa;">js/config.js</code> or configuration references and map your active endpoint:
        <pre style="background: #0f172a; padding: 15px; border-radius: 8px; border: 1px solid #1e293b; color: #e2e8f0; font-family: monospace;">window.APP_CONFIG = { BACKEND_URL: "http://127.0.0.1:8000" };</pre>
    </li>
    <li>Boot up the layout immediately by spinning up the file via VS Code <b>Live Server</b> extension or opening <code style="color: #34d399;">index.html</code> directly inside your default web browser window.</li>
</ol>
